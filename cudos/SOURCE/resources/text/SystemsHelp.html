Signal processing help

  1. Select an item from the list on the right hand side
  2. Press the 'Create' button below the list. The component you selected will appear in the diagram on the left.
  3. You can drag the item to move it around. On the bottom right is a box containing parameters for the selected component.
  4. The component's output is shown as a black dot on the right edge of the component
  5. If the component can have inputs, they shown as black dots on the left side of the component. Drag these black dots onto the output of another component to make a connection
  6. Pressing the start button on the bottom left of the screen begins calculation, and plots a graph of each component of type 'Output'
